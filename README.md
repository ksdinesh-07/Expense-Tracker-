# 💰 Expense Tracker

A simple and responsive **Expense Tracker** built using **HTML, CSS, and Vanilla JavaScript**. The application allows users to record income and expenses, calculate the current balance, and store transactions in the browser using Local Storage so that data persists even after refreshing the page.

## 🚀 Features

* ➕ Add income and expense transactions
* 🗑️ Delete transactions with confirmation
* 💰 Display current balance
* 📈 Track total income
* 📉 Track total expenses
* 💾 Save transactions using Local Storage
* 🔄 Automatically reload saved transactions
* 📱 Clean and responsive user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser Local Storage API

## 🎯 How It Works

1. Enter a transaction description.
2. Enter a positive amount for income or a negative amount for an expense.
3. Click the **Add Transaction** button.
4. The application updates:

   * Current Balance
   * Total Income
   * Total Expenses
   * Transaction History
5. All transactions are automatically saved to Local Storage and restored when the page is reopened.
